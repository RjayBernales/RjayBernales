### 🚀 Featured Project

#### 🌙 ELPMS - Environmental Light Pollution Monitoring System

> A PHP-based web application with IoT integration for monitoring and analyzing light pollution levels in the environment.

| Feature | Stack |
|---|---|
| Web App | PHP, HTML, CSS, JavaScript |
| Database | MySQL |
| IoT / Hardware | ESP32, Light Sensor - BH1750 chip |
| Auth & Roles | User · Manager · Admin |
| Charts | Chart.js (Line, Bar, Doughnut) |
